#### 任务2.2：添加能耗/成本多目标优化 ⭐⭐

**当前状态**: 仅优化Makespan（单目标）

**需要补充**:

**多目标适应度函数**:
```java
// 当前：单目标
double fitness = calculateMakespan(schedule);

// 改进：多目标加权
double makespan = calculateMakespan(schedule);
double energy = calculateEnergy(schedule);  // 新增
double cost = calculateCost(schedule);      // 新增

double fitness = ALPHA * makespan + BETA * energy + GAMMA * cost;
// 权重：ALPHA=0.6, BETA=0.3, GAMMA=0.1
```

**能耗模型**:
```
Energy = Σ(VM_Power × VM_Runtime)
VM_Power = Base_Power + Dynamic_Power × CPU_Utilization
```

**成本模型**:
```
Cost = Σ(VM_Hourly_Rate × VM_Runtime)
```

**实施步骤**:
1. 添加 `EnergyCalculator.java` 和 `CostCalculator.java`
2. 修改 `LSCBO_Broker_Fixed.java` 集成多目标
3. 对比实验：单目标 vs 多目标

**预期结果**:
- 能耗降低：15-25%（vs 单目标Makespan）
- 成本降低：10-20%
- Makespan略增：<5%（可接受权衡）

**提升价值**: 云任务调度的实际应用价值，符合Q2期刊的工程导向。
